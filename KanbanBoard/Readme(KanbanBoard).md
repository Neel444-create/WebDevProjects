Simple Kanban Board
A lightweight, responsive, and dependency-free Kanban Board web application built using native HTML5 Drag and Drop API, CSS3, and vanilla JavaScript.

This project allows users to visually manage tasks by dragging cards between To Do, In Progress, and Done columns.

🚀 Features
Native Drag and Drop: Leverages the built-in HTML5 Drag and Drop API for smooth and lightweight interactions without external libraries.

Visual Drop Indicators: Columns dynamically highlight when a task card is dragged over them, providing instant visual feedback.

Responsive Layout: Automatically scales from a multi-column desktop view to a clean, stacked layout on mobile devices.

Modern UI: Clean aesthetic featuring smooth scaling transitions, custom grab cursors, and subtle shadows.
